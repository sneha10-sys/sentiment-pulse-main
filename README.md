# 📊 SentimentScope API

A FastAPI-based backend for **sentiment analysis of text data** using NLP techniques.  
It supports single text analysis and batch processing via file uploads (CSV/TXT).  
The system combines multiple sentiment models for improved accuracy and extracts meaningful keywords.

---

## 🚀 Features

- 🔍 Real-time sentiment analysis (Positive / Negative / Neutral)  
- 🧠 Hybrid NLP model (VADER + TextBlob ensemble)  
- 📌 Keyword extraction using TF-IDF  
- 📁 Batch processing via file upload (CSV/TXT)  
- ⚡ FastAPI-based high-performance backend  
- ✅ Strict data validation using Pydantic  
- 🌐 CORS enabled for frontend integration  

---

## 🏗️ Tech Stack

- **Backend Framework:** FastAPI  
- **Server:** Uvicorn  
- **NLP Libraries:** VADER, TextBlob, NLTK  
- **ML Utilities:** Scikit-learn, Pandas, NumPy  
- **Validation:** Pydantic  
- **Language:** Python 3.10+
